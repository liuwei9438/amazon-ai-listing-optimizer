# Amazon AI Listing Optimizer V2.2.1

## 当前目标

本版本只建立并验收稳定数据层：

- 读取 `.xlsx`
- 识别常见字段
- 诊断图片链接列和嵌入图片
- 原样导出上传文件
- 用 SHA-256 验证导出与输入字节级一致

因此，本阶段不会丢失：

- 列名和列顺序
- 单元格内容
- 格式、公式、超链接
- 嵌入图片
- 未知或自定义字段

## 暂未启用

- AI 产品理解
- 标题、短标题、五点和详情生成
- 多语言生成
- 图片优化和 Cloudinary

## V1.3.2 图片基线

`image/v132_image_pipeline.py` 保存 V1.3.2 图片处理基线，并处于冻结状态。
后续版本只接入，不重写；除非用户明确要求修改图片效果。

## 运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 验收标准

上传真实 Excel 后：

1. 页面显示行数、列数、图片对象数量；
2. 下载测试文件；
3. 文件可以正常打开；
4. 原始字段、格式和图片完全保留；
5. 页面显示“导出文件与原文件完全一致”。
